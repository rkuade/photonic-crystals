# photonic-crystal

Code to optimize for a structure exhibiting a photonic bandgap.

To run the code, enter "sbatch run_rampQabs_AllPolarizations_ceviche_LDOSmin_dipole_oneside_notes_embed_PB.sh" into the command line of a computing cluster.

Requires: ceviche, NLopt, NumPy, Autograd, Matplotlib, and Python 3
